## 📚 3. 各模块简介

按仓库顶层目录的顺序，简要介绍各模块:

1. `01.算法学习笔记` — 汇集算法与数据结构的学习笔记与题解，包含课程笔记、练习与代码示例。
2. `05.CS学习之路` — 记录大学专业课程笔记（如数据结构、操作系统、计算机网络等）。
3. `10.杂想集` — 个人随笔、AI 作品、日记与灵感记录，内容较为零散与创意性强。

## 🚀 4. 快速开始

### 浏览笔记

1. 克隆此仓库：
```bash
git clone https://github.com/sunyzhi55/note-harbor-backup.git
cd note-harbor-backup
```

2. 浏览相应目录下的笔记文件

### 添加笔记

在对应的模块目录下创建新的笔记文件：

- 算法笔记 → `01.算法学习笔记/`
- 专业课笔记 → `05.CS学习之路/`
- 杂想内容 → `10.杂想集/`

### 推荐的笔记格式

- 使用 Markdown 格式编写
- 为每个主题创建独立文件或文件夹
- 使用清晰的标题和目录结构
- 包含代码示例和图表说明

## 📝 5. 笔记命名规范

- 使用中文或英文清晰描述笔记主题，添加序号以便排序
- 示例：`01.快速排序.md`
- 对于关联的多个文件，可创建子目录管理
