## 💡 记录建议

为了保持笔记的高质量与系统性，建议遵循以下原则：

- **分层整理**：按课程或语言单独创建文件夹，每个主题一个文件，方便后续查阅。
- **课程笔记**：建议保留 **课程大纲**  **关键公式与概念**  **实验总结**  **复习要点**。
- **语言笔记**：建议包含 **知识点总结**  **代码示例**  **常见错误**  **进阶拓展**。
- **结构化**：每篇笔记保留清晰的章节结构，重点概念可单独摘出做总结页，便于复习。

## 📋 目录使用原则

1. **优先归档**：新内容优先放入对应子目录，保持根目录整洁。
2. **分离理论与实践**：课程笔记、语言学习、项目复盘可分开保存，方便针对性查阅。
3. **持续完善**：定期回顾旧笔记，补充遗漏的知识点和实战经验。
4. **实验总结**：课程实验和项目实践的总结要及时归档，积累工程经验。
